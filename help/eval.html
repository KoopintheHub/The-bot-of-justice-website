How to unrestrict the eval command.

First, you will need to goto bot options by typing in &options and then access the restrictions page
by typing in &options page-restrictions. Then
type in &options eval-restrict=false. Now you will see a message saying:

"This will unrestrict the eval command.
WARNING: THIS IS A RISKY COMMAND IT MIGHT MESS UP THE SERVER
DEPENDING ON WHAT CODE YOUR TYPING IN!
Do you want to continue? (&yes/&no)"


Now if you change your mind, type in: &no.
Type in: &yes if you want to unrestrict the command.
